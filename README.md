#Proyecto de Práctica .Net Framework C#

Introducción
------------
Este proyecto es un trabajo final del Nivel 2 del curso de Maxi programa. 
La Aplicación  gestiona los artículos de un comercio.
Si te interesa el curso te dejo su página para que veas toda la info:  [MaxiPrograma](https://maxiprograma.com/mim "MaxiPrograma")

- Windows Forms
- Paradigma de programación orientada a objetos.
-  Integración con una base de datos SQL server
